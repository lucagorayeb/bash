# O comando kill

Serve para cancelar os próprios processos que estão sendo executados.

O super usuário  consegue matar processos de outros usuários.

Parâmetros do comando kill:

    Sinal : O sinal é passado para que o comando execute uma função 
            com o processo que vai ser informado como -15 para terminar,
            -9 para forçar ele a terminar (matar o processo).

    PID   : O número do processo que deve ser passado para que o comando
            kill execute a ação.
