# O comando gzip 

O gzip é um compactador de arquivos. É o mais utilizado pelos os usuários
de software livre.

Opções mais utilizadas do gzip:
    
    -c : Escreve na saída padrão, sem modificar o arquivo especificado.

    -f : Força a compactação. Não compacta se o arquivo especificado não
         pudar ser compactado ou se um arquivo compactado correspondente 
         (extensão .gz) existir.

    -v : Mostra informações sobre a porcentagem de compactação de cada arquivo.

    -r : Comprime também o conteúdo dos subdiretórios (recursivo).

    -d : Expande arquivos compactados (decompress).

# O comando guzip 

O gunzip é um desconpactador de arquivos.

Opções mais utilizadas do gunzip:

    -c : Escreve na saída padrão, sem gerar arquivo descompactado.

    -v : Mostra informções sobre a porcentagem de compactação de 
         cada arquivo.
